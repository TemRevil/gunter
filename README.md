# Modern Mechanic System (Lack)

A specialized management system for mechanical workshops and spare part stores. Built with modern web technologies, focusing on speed, security, and a premium user experience.

## ğŸš€ Key Features

- **Multi-language Support:** Full Arabic and English interface with automatic RTL/LTR layout switching.
- **Operations Log:** Track daily sales, payments, and debts with a sleek, searchable interface.
- **Inventory Management:** Manage spare parts with low-stock alerts and history tracking.
- **Customer Management:** Keep track of customer data, transaction history, and detailed debt balances.
- **Financial Reports:** Automatic daily session closing with cash reconciliation, differences report, and printing support.
- **Security:** Tiered access with System and Admin passwords to protect sensitive actions like deletions and settings changes.
- **Responsive Design:** Premium UI that works beautifully on desktops, tablets, and mobile devices.
- **Offline First:** Data is stored locally in the browser, ensuring the system works without a constant internet connection.
